1. Créez d’abord une brique de contenu.
2. Ajoutez ensuite une version de cette brique.

4. Ajoutez une ou plusieurs étapes à ce scénario.
5. Les types de déclencheur et les événements sont gérés automatiquement par le système et sont visibles ici uniquement pour consultation.

Créer une brique de contenu

Une brique est un morceau de message réutilisable : alerte, conseil, rappel, conclusion…
Nom unique utilisé par le système.
Exemple : ne montrer ce contenu qu’une fois par événement, avec un délai de répétition de 7 jours.

Briques existantes

Créer une version de contenu

Une même brique peut avoir plusieurs versions : brouillon, publiée, archivée…
Vous pouvez utiliser des variables comme {lieu}, {source_url}, {territory}…
